# 1. 프로젝트 개요

본 시스템은 대산의 건축자재(석고보드 등) 출고 시, **비용 최적화**와 **법적 안전성**을 동시에 충족하는 차량 선택 및 적재 배치를 자동화하는 것을 목적으로 한다.

Woodcutter 프로젝트에서 축적된 판재 최적화 알고리즘을 물류 적재함 공간(2.5D)으로 확장하며, 팔레트 단위·물리 제약·비용 최적화를 기반으로 혼적 제약과 위험도 레벨링을 통해 실제 현장 의사결정을 지원한다.

---

# 2. 버전 통합 개요 (V10.1)

V7과 V10 각각의 핵심 강점을 통합하고, 실무 적용 공백을 보완한 버전이다.

| 출처 | 통합 내용 |
|------|-----------|
| V7 유지 | 혼적 제약 로직, 상단·취약 자재 위험 판단, 공간 적재율 전 구간 정의, 운영 체크리스트 |
| V10 유지 | Pre-Check 배차 선행 검증, 법적 축중 10톤 Constraint, OR-Tools MIP, COG 수식 명시, 현장 매뉴얼 자동 생성 |
| V10.1 신규 | OR-Tools 환경 요건 명시, 2.5D 처리 한계 기준 정의 |

# 3. 시스템 흐름

```
사용자 직접 입력 (자재명, 수량, 규격)
  → 단위 환산 (낱장 → 팔레트)
  → Pre-Check 시뮬레이션 (2.5D Bin Packing)
  → MIP 기반 차량 선택 (OR-Tools)
  → 최종 적재 배치 + COG / 축중 계산
  → 혼적 제약 검증
  → 위험도 레벨 평가 (7개 항목)
  → 결과 출력 (상차 지시서 + 현장 매뉴얼 자동 생성)
```

# 4. 알고리즘 설계

## 4.1 단위 환산

자재별 DB를 기반으로 낱장 → 팔레트(Pallet) 단위 변환 후 총 중량 및 부피를 계산한다.

## 4.2 Pre-Check 시뮬레이션

배차 확정 **전**, 후보 차량 적재함에 2.5D Bin Packing을 가상 수행하여 물리적 불가 상황을 원천 차단한다.

- 바닥면 배치 우선, 동일 규격 자재의 수직 적재(Stacking) 허용
- 불가 판정 시 즉시 차량 변경 또는 분할 배차 안내

**처리 한계 기준**

| 조건 | 권장 한계 |
|------|-----------|
| 자재 종류 수 | 20종 이하 |
| 팔레트 총 수 | 50개 이하 |
| 후보 차량 수 | 10대 이하 |

> 한계 초과 시 OR-Tools Time Limit 파라미터로 조기 종료 후 근사해 반환.

## 4.3 MIP 기반 차량 선택

Google OR-Tools를 활용해 최저 비용 차량 조합을 산출한다.

**OR-Tools 환경 요건**

| 항목 | 요건 |
|------|------|
| 설치 방식 | `pip install ortools` |
| Python 버전 | 3.9 이상 |
| 서버 최소 스펙 | CPU 4코어, RAM 8GB |
| 라이선스 | Apache 2.0 (상업 이용 가능) |
| 권장 Time Limit | 30초 (실시간 출고 환경 기준) |

## 4.4 최종 배치 및 축중 계산

하중 중심(COG)을 고려한 최종 적재 위치를 확정한다.

**축중 계산 수식:**

$$W_{axle} = \sum \left( W_i \times \frac{L - d_i}{L} \right)$$

- $L$: 축간거리
- $d_i$: 각 자재 중심의 전축으로부터의 거리
- $W_i$: 각 자재 중량

# 5. 위험도 레벨 체계

| 레벨 | 명칭 | 의미 |
|------|------|------|
| 1 | 정상 (Safe) | 기준치 이하, 이상 없음 |
| 2 | 주의 (Caution) | 경계 구간, 모니터링 필요 |
| 3 | 위험 (Danger) | 즉각 조치 권고 |
| 4 | 불가 (Critical) | 적재 중단, 차량 변경 필수 |

---

# 6. 위험 판단 기준 (7개 항목)

## 6.1 과적 위험 (하중)

| 구간 | 레벨 |
|------|------|
| 85% 이하 | 정상 |
| 85 ~ 95% | 주의 |
| 95 ~ 100% | 위험 |
| 100% 초과 | 불가 |

## 6.2 법적 축중 제한

| 조건 | 레벨 |
|------|------|
| 모든 축 10톤 이하 | 정상 |
| 어느 한 축이라도 10톤 초과 | 불가 (즉시 차단) |

> 근거: 도로교통법 축당 하중 10톤 제한

## 6.3 공간 적재율

| 구간 | 레벨 |
|------|------|
| 70% 이하 | 여유 (정상) |
| 70 ~ 85% | 적정 (정상) |
| 85 ~ 95% | 주의 (상차 난이도 상승) |
| 95% 초과 | 위험 |

## 6.4 전후 하중 편차

| 구간 | 레벨 |
|------|------|
| 10% 이하 | 정상 |
| 10 ~ 20% | 주의 |
| 20 ~ 30% | 위험 (재배치 권고) |
| 30% 초과 | 불가 검토 |

## 6.5 좌우 하중 편차

| 구간 | 레벨 |
|------|------|
| 10% 이하 | 정상 |
| 10 ~ 20% | 주의 |
| 20 ~ 30% | 위험 |
| 30% 초과 | 불가 |

## 6.6 상단 적재 위험

| 조건 | 레벨 |
|------|------|
| 정상 적재 (중량 하단, 경량 상단) | 정상 |
| 상단 비중 증가 (상단 하중 > 전체의 40%) | 주의 |
| 상단 과적 | 위험 |

## 6.7 취약 자재 위험

| 조건 | 레벨 |
|------|------|
| 취약 자재 최상단 배치 | 정상 |
| 취약 자재 하단 압박 | 위험 |

# 7. 혼적 제약

## 7.1 적재 방향 제약

방향이 지정된 자재는 지정 방향 외 배치 불가. 방향 혼합 시 별도 구획 필요.

## 7.2 취급 등급 제약

| 등급 | 내용 |
|------|------|
| A (취약) | 단독 구획 또는 최상단 배치 필수 |
| B (일반) | 표준 적재 허용 |
| C (중량) | 하단 우선 배치 |

## 7.3 혼적 그룹 제약

지정된 혼적 불가 그룹 간 동일 적재함 배치 금지.

> 예: 석고보드(취약, A등급) + 중량 철물류(C등급) → 혼적 불가

## 7.4 동일 규격 우선 배치

동일 규격 자재는 같은 팔레트 또는 인접 구획에 우선 배치하여 수직 적재 효율을 극대화한다.

# 8. 위험도 연동 현장 운영 매뉴얼

시스템이 계산한 위험 레벨에 따라 현장 작업자 결속(Strap) 매뉴얼을 자동 생성한다.

| 위험 레벨 | 현장 필수 조치 사항 |
|-----------|---------------------|
| 정상 (Safe) | 표준 결속 및 상차 사진 촬영 후 출고 |
| 주의 (Caution) | 고탄성 스트랩 2중 결속 및 측면 고임목 설치 필수 확인 |
| 위험 (Danger) | 관리자 최종 승인, 저속 운행(60km/h 이하) 지침 하달 |
| 불가 (Critical) | 배차 및 상차 즉시 중단 — 차량 톤수 상향 또는 분할 배차 |
